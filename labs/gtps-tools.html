GTPS Tools

Release

A suite of open-source tools for private game servers.

← Back to Labs

GTPS Tools is a collection of command-line utilities and web-based tools designed for players and administrators of private game servers, with a primary focus on Growtopia Private Servers (GTPS).

Features

  • Item ID resolver and information lookup.
  • World data parser and visualizer (conceptual).
  • Save data editor with safety checks.
  • Packet manipulation utilities for server interaction analysis (for educational purposes).
  • A web dashboard for managing some of these tools with a user-friendly interface.

Current Status: Release

Several tools within the GTPS Tools suite are considered stable and have been released. Ongoing development focuses on:

  • Adding support for more game server types.
  • Improving the accuracy and features of existing tools.
  • Enhancing the web dashboard with more functionalities.
  • Maintaining compatibility with the latest game versions.

All tools are open-source and contributions are welcome!

Technologies Used

  • Python (for CLI tools)
  • JavaScript (for web tools)
  • Flask / FastAPI (for backend of web tools)
  • HTML/CSS
